# 员工薪资计算规则

## 数据来源

### 源文件1：员工信息 (01_员工信息.xlsx)
- Sheet名：员工信息
- 列：工号、姓名、部门、基本工资

### 源文件2：考勤数据 (02_考勤数据.xlsx)
- Sheet名：考勤记录
- 列：工号、出勤天数、加班小时

### 源文件3：绩效数据 (03_绩效数据.xlsx)
- Sheet名：绩效评定
- 列：工号、绩效等级、绩效系数

## 数据关联规则
- 主键：工号
- 所有表通过"工号"字段进行关联

## 计算规则

### 1. 日薪计算
```
日薪 = 基本工资 / 21.75
```
说明：21.75为月均工作日

### 2. 出勤工资
```
出勤工资 = 日薪 × 出勤天数
```

### 3. 加班费计算
```
加班费 = 日薪 / 8 × 加班小时 × 1.5
```
说明：
- 8为日标准工时
- 1.5为加班费倍率

### 4. 应发工资
```
应发工资 = 出勤工资 + 加班费
```

### 5. 绩效奖金
```
绩效奖金 = 基本工资 × 绩效系数
```
绩效系数对照：
- S级：1.5
- A级：1.2
- B级：1.0
- C级：0.8

### 6. 社保扣款
```
社保扣款 = 基本工资 × 10.5%
```

### 7. 实发工资
```
实发工资 = 应发工资 + 绩效奖金 - 社保扣款
```

## 输出要求

### 输出文件：薪资汇总表.xlsx
- Sheet名：薪资明细
- 输出列（按顺序）：
  1. 工号
  2. 姓名
  3. 部门
  4. 基本工资
  5. 出勤天数
  6. 加班小时
  7. 绩效等级
  8. 绩效系数
  9. 应发工资
  10. 绩效奖金
  11. 社保扣款
  12. 实发工资
### 特殊规则
- 出勤天数如果大于20，此格子标注红色
### 数值精度
- 所有金额保留2位小数
- 使用四舍五入

### 排序规则
- 按工号升序排列
