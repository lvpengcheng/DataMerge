# 数据清洗功能最终确认

## 问题
用户担心：如果存在数据清洗逻辑，主表数据直接复制时是否也应用了清洗逻辑？

## 答案
✅ **是的，主表数据已经应用了清洗逻辑**

## 工作流程说明

### 完整的数据流

```
原始Excel文件
    ↓
[步骤1] load_source_data()
    ↓
source_data (清洗前的DataFrame)
    ↓
[步骤1.5] clean_source_data(source_data)  ← AI生成的清洗函数
    ↓
source_data (清洗后的DataFrame)
    ↓
[步骤2] write_source_sheets(wb, source_data)
    ↓
source_sheets (清洗后的DataFrame + worksheet)
    ↓
[步骤3] fill_result_sheets(wb, source_sheets, ...)  ← AI生成的填充函数
    ↓
    ├─ 主表数据：直接复制 source_sheets 中的 DataFrame（已清洗）
    └─ 非主表数据：VLOOKUP 引用 source_sheets 中的 worksheet（已清洗）
```

### 关键点

1. **数据清洗时机**：在步骤1.5，所有源数据都会经过 `clean_source_data` 函数清洗
2. **主表数据复制**：在步骤3中，主表数据直接复制的是 `source_sheets` 中的 DataFrame，这些数据已经是清洗后的
3. **VLOOKUP引用**：VLOOKUP 引用的源数据 worksheet 也是清洗后写入Excel的，已过滤掉不符合规则的记录

### 代码示例

#### 主函数（固定模板）
```python
def main():

    fill_result_sheets(wb, source_sheets, ...)
```

#### AI生成的clean_source_data函数示例
```python
def clean_source_data(source_data):
    """应用数据清洗规则过滤源数据"""
    cleaned_data = {}

    # 清洗员工基础表
    if "员工基础表_员工信息" in source_data:
        df = source_data["员工基础表_员工信息"]["df"].copy()
        original_count = len(df)

        # 1. 去除重复工号
        df = df.drop_duplicates(subset=['工号'], keep='first')

        # 2. 过滤空值
        df = df.dropna(subset=['工号', '姓名'])

        # 3. 过滤离职员工
        if '员工状态' in df.columns:
            df = df[df['员工状态'] == '在职']

        cleaned_count = len(df)
        print(f"员工基础表: 原始{original_count}条 → 清洗后{cleaned_count}条")

        cleaned_data["员工基础表_员工信息"] = {
            "df": df,
            "columns": list(df.columns)
        }

    # 清洗其他表...

    return cleaned_data
```

#### AI生成的fill_result_sheets函数示例
```python
def fill_result_sheets(wb, source_sheets, salary_year=None,
                       salary_month=None, monthly_standard_hours=174):
    """填充结果sheet"""

    # 选择主表（员工数最多的表）
    main_table_name = "员工基础表_员工信息"
    main_df = source_sheets[main_table_name]["df"]  # ← 这是清洗后的DataFrame

    # 创建结果sheet
    result_ws = wb.create_sheet("薪资报表")

    # 写入表头
    headers = ["工号", "姓名", "身份证号", "部门", "职位", "基本工资", ...]
    for col_idx, header in enumerate(headers, 1):
        result_ws.cell(1, col_idx, header)

    # 填充数据
    for r, row in enumerate(main_df.itertuples(index=False), 2):
        # 主表数据：直接复制（已清洗）
        result_ws.cell(r, 1, row.工号)      # ← 清洗后的数据，无重复、无空值、无离职
        result_ws.cell(r, 2, row.姓名)
        result_ws.cell(r, 3, row.身份证号)
        result_ws.cell(r, 4, row.部门)
        result_ws.cell(r, 5, row.职位)
        result_ws.cell(r, 6, row.基本工资)

        # 非主表数据：VLOOKUP引用（引用的也是清洗后的数据）
        sn_attend = source_sheets["考勤表_考勤记录"]["ws"].title
        result_ws.cell(r, 7).value = f"=IFERROR(VLOOKUP(A{r},'{sn_attend}'!$A:$E,3,FALSE),0)"
        # ↑ 引用的考勤表worksheet也是清洗后的，已过滤空值、重复工号等
```

## 提示词中的明确说明

### 1. 已有变量说明
```markdown
## 已有变量
- wb: openpyxl Workbook
- source_data: {"文件名_sheet名": {"df": DataFrame, "columns": [列名]}} （清洗前）
- source_sheets: {"文件名_sheet名": {"df": DataFrame, "ws": worksheet}} （清洗后，写入Excel）
```

### 2. 执行流程说明
```markdown
## 执行流程（固定代码已处理）
1. ✅ 源数据已加载到 source_data 字典（DataFrame格式）
2. 🎯 你的任务1：生成clean_source_data函数，应用清洗规则过滤数据
3. ✅ 清洗后的数据会写入Excel（供公式引用）
4. ✅ 参数sheet已创建
5. 🎯 你的任务2：生成fill_result_sheets函数，创建结果sheet并填充公式
```

### 3. 核心规则说明（新增）
```markdown
## 【规则2】VLOOKUP是唯一跨表取数方式
- 主表数据：直接复制,不能用VLOOKUP（会导致性能问题）
- ⚠️ **重要**：主表数据复制时使用的是source_sheets中的数据，这些数据已经过clean_source_data函数清洗，符合所有数据清洗规则
- 非主表数据：**必须且只能**用VLOOKUP，禁止任何其他方式
- ⚠️ **重要**：VLOOKUP引用的源数据sheet也是清洗后的数据，已过滤掉不符合规则的记录
```
