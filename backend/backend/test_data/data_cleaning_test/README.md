## 计算规则说明

### 迟到扣款
```
迟到扣款 = 迟到次数 × 50
```

### 请假扣款
```
请假扣款 = 请假天数 × (基本工资 / 21.75)
```

### 应发工资
```
应发工资 = 基本工资 - 迟到扣款 - 请假扣款
```

### 实发工资
```
实发工资 = 应发工资 - 社保个人缴纳
```
