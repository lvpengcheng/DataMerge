## 一、数据来源

### 源文件1：员工基础表

- 读取方式：导入
- 用途：获取员工基础信息

| 列名 | 是否需要存储 | 与报表项关系 |
|------|:------------:|-------------|
| 工号 | 是 | 报表项"工号" |
| 姓名 | 是 | 报表项"姓名" |
| 身份证号 | 是 | 报表项"身份证号" |
| 部门 | 是 | 报表项"部门" |
| 职位 | 是 | 报表项"职位" |
| 基本工资 | 是 | 报表项"基本工资" |
| 入职日期 | 是 | 报表项"入职日期" |
| 员工状态 | 是 | 报表项"员工状态" |

**导入校验规则：**

- "工号"或"姓名"为空，导入失败
- "员工状态"必须为"在职"或"离职"

## 二、输出报表

### 输出文件：员工薪资报表
- Sheet名：薪资报表

| 列名 | 计算规则 |
|------|----------|
| 工号 | 来自员工基础表 |
| 姓名 | 来自员工基础表 |
| 身份证号 | 来自员工基础表 |
| 部门 | 来自员工基础表 |
| 职位 | 来自员工基础表 |
| 基本工资 | 来自员工基础表 |
| 出勤天数 | 来自考勤表，如果考勤表中没有该员工，则为0 |
| 迟到次数 | 来自考勤表，如果考勤表中没有该员工，则为0 |
| 请假天数 | 来自考勤表，如果考勤表中没有该员工，则为0 |
| 迟到扣款 | 迟到次数 × 50 |
| 请假扣款 | 请假天数 × (基本工资 / 21.75) |
| 应发工资 | 基本工资 - 迟到扣款 - 请假扣款 |
| 社保个人缴纳 | 来自社保表，如果社保表中没有该员工，则为0 |
| 实发工资 | 应发工资 - 社保个人缴纳 |

## 三、数据清洗规则

- 【员工基础表】，工号重复的数据不能作为计薪雇员。

- 【员工基础表】，工号或姓名为空的不能作为计薪雇员。

- 【员工基础表】，员工状态必须为"在职"，否则不能作为计薪雇员。

- 【考勤表】，工号或姓名为空的不能参与考勤统计。

- 【考勤表】，工号对应的姓名必须与员工基础表中工号对应的姓名一致，否则不能参与考勤统计。

- 【考勤表】，工号重复的不能参与考勤统计。

- 【社保表】，身份证号重复的记录对应的员工不计入社保统计。

- 除员工基础表外，在其他所有导入表中均无记录的工号，不计入计薪雇员。

- 所有导入项均为0且计算结果均为0的工号，不参与报表统计。

---

## 四、警告信息

社保表中的身份证号必须在员工基础表中存在，否则需提示"社保表中身份证号未匹配到员工基础表"的记录。

员工基础表中的部门必须在部门信息表中存在，否则需提示"员工基础表中部门未匹配到部门信息表"的记录。

考勤表中的工号必须在员工基础表中存在，否则需提示"考勤表中工号未匹配到员工基础表"的记录。
