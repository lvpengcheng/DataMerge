# 修复多步分析模式的数据清洗功能

## 问题发现

用户发现在生成的代码中：
1. 主函数中有调用 `clean_source_data` 的代码
2. 但是 `clean_source_data` 函数本身没有生成
3. AI只是在 `fill_result_sheets` 函数内部对主表进行了过滤：`main_df = main_df[main_df["部门"] != "市场部"].copy()`

## 根本原因

系统有两种生成模式：

### 1. 单次生成模式 (`use_multi_step = false`)
- 使用 `generate_formula_mode_prompt` 方法
- ✅ 已经包含数据清洗规则和 `clean_source_data` 函数要求
- ✅ 提示词正确

### 2. 多步分析模式 (`use_multi_step = true`) ← **问题所在**
- 使用 `generate_multi_step_prompts` 方法
- ❌ 没有提取数据清洗规则
- ❌ 没有要求AI生成 `clean_source_data` 函数
- ❌ 提示词中仍然是旧版本的说明

rex104的训练使用的是多步分析模式，所以AI没有生成 `clean_source_data` 函数。

## 解决方案

修改 `generate_multi_step_prompts` 方法，添加数据清洗功能：

### 修改1：提取数据清洗规则

```python
# 在方法开头添加
extracted_rules = self.rule_extractor.extract_rules(rules_content)
data_cleaning_rules_text = ""

if extracted_rules["data_cleaning_rules"]:
    data_cleaning_rules_text = "\n## 数据清洗规则（在clean_source_data中应用）\n"
    data_cleaning_rules_text += "⚠️ 在将源数据写入Excel之前，必须先应用以下清洗规则过滤数据：\n\n"
    for i, rule in enumerate(extracted_rules["data_cleaning_rules"], 1):
        if rule['original_text'] and not rule['original_text'].startswith('--'):
            data_cleaning_rules_text += f"{i}. {rule['original_text']}\n"
    data_cleaning_rules_text += "\n**实现方式**：生成clean_source_data函数，对每个DataFrame应用清洗逻辑后返回清洗后的数据。\n"
```

## AI生成的错误行为分析

### 当前AI的做法（错误）

```python
def fill_result_sheets(wb, source_sheets, ...):
    # 在fill_result_sheets内部过滤主表
    main_df = source_sheets[main_key]["df"]
    main_df = main_df[main_df["部门"] != "市场部"].copy()  # ← 只清洗了主表

    # 问题：
    # 1. 只清洗了主表，其他表（考勤表、社保表等）没有清洗
    # 2. VLOOKUP引用的源数据sheet仍然包含未清洗的数据
    # 3. 清洗逻辑分散在fill_result_sheets中，不符合设计
```

### 期望的做法（正确）

```python

    cleaned_data = {}

            "columns": list(df.columns)
        }

def fill_result_sheets(wb, source_sheets, ...):

    main_df = source_sheets[main_key]["df"]  # ← 已经是清洗后的数据

    # 创建结果sheet并填充
    ...
```

## 为什么AI会犯这个错误？

1. **提示词不明确**：旧版提示词没有明确要求生成两个函数
2. **变量说明混淆**：提示词中说 `source_sheets` 已经可用，AI以为不需要清洗
3. **缺少清洗规则章节**：没有专门的数据清洗规则章节，AI不知道要清洗什么
4. **缺少任务分解**：没有明确说明"任务1：生成clean_source_data"和"任务2：生成fill_result_sheets"

## 修改后的效果

修改后，AI应该能够：
1. ✅ 生成独立的 `clean_source_data` 函数
2. ✅ 在函数中应用所有数据清洗规则
3. ✅ 清洗所有相关的表（不只是主表）
4. ✅ 在 `fill_result_sheets` 中直接使用清洗后的数据
5. ✅ 确保VLOOKUP引用的也是清洗后的数据
